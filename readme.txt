Installation:
---------------
Installable file name is  "ipaytotal_hosted.ocmod.zip"

1. Please Go to Extensions -> Extension Installer
2. Upload "ipaytotal_hosted.ocmod.zip" and click on continue.
3. Now install from Admin -> Extensions -> Filter by Payment -> iPayTotal Hosted

Configuration:
--------------

1. Please navigate to Admin -> Extensions -> Filter by Payment -> iPayTotal Hosted
2. Now click the Edit button of the iPayTotal Hosted row
3. Enter your iPaytotal API Key and change module status to enable
4. Click the Save button to store your setting
